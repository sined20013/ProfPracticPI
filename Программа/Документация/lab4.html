Feedback

Описание: Класс для реализации обратной связи, даёт возможность для системы отправлять отзывы на сообщения пользователя

Автор: Харченко Денис Александрович

  • Метод: feedIcon

    Описание: Отзыв в виде иконки/смайлика

  • Метод: feedText

    Описание: Отзыв в виде текстового сообщения

Interface

Описание: Класс для выбора интерфейса, позволит пользователям взаимодействовать с системой

Автор: Харченко Денис Александрович

  • Метод: TelephoneInterface

    Описание: Интерфейс для телефонов

  • Метод: KomputerInterface

    Описание: Интерфейс для ПК

  • Метод: WearInterface

    Описание: Интерфейс для периферии
